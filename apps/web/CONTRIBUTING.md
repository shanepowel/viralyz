# Copy standards

- Sentence case everywhere, including buttons and nav.
- Buttons name the action's result: `Score your video, free`, `Get notified at launch`, never `Submit` / `Learn more`.
- Same verb through a flow: `Send message` → success `Message sent`.
- Numbers always via `Num` / formatters; en-GB spelling; £ only.
- No claim without evidence: any sentence containing a statistic must reference `lib/site.ts` stats — if the stat is `null`, the sentence does not ship.
- Errors state what happened + what to do; empty states invite one action.
